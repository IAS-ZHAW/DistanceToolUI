Matrix abspeichern

Bitte legen Sie fest wo Sie die Matrix abspeichern wollen.

Ausgabedatei

Dies ist die Datei, welche die Daten enthält. Jede Zeile muss einem Datensatz entsprechen und jede Spalte einer Frage bzw. einem Merkmal. Die einzelnen Werte müssen mit einem Trennzeichen getrennt werden.

Trennzeichen

Konfiguration des Zeichens mit dem die einzelnen Werte voneinander getrennt sind. Es muss sich um ein einzelnes Zeichen handeln. In der Tabelle kann kontrollliert werden, ob die Werte korrekt in verschiedene Spalten übetragen wurden.

Genauigkeit

Anzahl Nachkommastellen mit der die Distanzen in der Datei gespeichert werden. Diese Angabe hat nichts mit der Genauigkeit der Berechnungen zu tun, es handelt sich nur um eine abschliessende Rundung.
