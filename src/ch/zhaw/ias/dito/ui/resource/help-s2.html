Fragenspezifikation

Bitte spezifizieren Sie die Fragen mit welchen Sie die Berechnungen vornehmen möchten.

Fragentabelle

Eine Auflistung aller importierten Merkmale.

  • Zellen mit gelbem Hintergrund lassen sich editieren
  • Durch Klicken auf die Spaltentitel können die Fragen umsortiert werden
  • Durch Doppelklicken auf eine Zelle mit weissem Hintergrund, wird ein Histogramm der entsprechenden Frage geöffnet
  • Mit dem Button oben rechts der Tabelle können Spalten ein-/ausgeblendet werden

Titel

Der Titel dient nur dem Benutzer dazu, die Fragen unterscheiden zu können. Falls keine Titel importiert wurden, wird automatisch der Titel "Question #Nummer" vergeben. Der Titel nimmt keinen Einfluss auf den Algorithmus.

Farben ORDINALerkmale mit mehr als zwei Ausprägungen, die sich aber in eine Reihenfolge bringen lassen. METRICerkmale mit sehr vielen/beliebig vielen Ausprägungen, die sich in eine Reihenfolge bringen lassen. z.B. Grösse eines Menschen
